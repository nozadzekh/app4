Smart Menu - Digital Culinary Assistant
Smart Menu არის თანამედროვე Android აპლიკაცია, რომელიც მომხმარებელს სთავაზობს 20-ზე მეტ პრემიუმ ხარისხის რეცეპტს მთელი მსოფლიოდან. აპლიკაცია შექმნილია სისწრაფეზე, ოფლაინ ხელმისაწვდომობასა და მომხმარებლის საუკეთესო გამოცდილებაზე (UX) ორიენტირებით.
მთავარი ფუნქციები
Offline-First: აპლიკაცია მუშაობს სრულყოფილად ინტერნეტის გარეშე, ლოკალური მონაცემთა ბაზის წყალობით.
HD ვიზუალიზაცია: თითოეული კერძი წარმოდგენილია მაღალი ხარისხის ფოტოებით.
ჭკვიანი ფილტრაცია: რეალურ დროში ძებნა და კატეგორიზაცია (ქართული, იტალიური, აზიური, დესერტები).
დეტალური ინსტრუქციები: ინგრედიენტების ზუსტი ჩამონათვალი, მომზადების დრო და სირთულის დონეები.
ტექნოლოგიური სტეკი (Tech Stack)
ენა: Kotlin
არქიტექტურა: MVVM (Model-View-ViewModel)
UI: Material 3, RecyclerView (ListAdapter + DiffUtil)
Async & Reactive: Kotlin Coroutines, Flows & StateFlow, LiveData
სურათების მართვა: Glide (Image Caching & Processing)
ნავიგაცია: Jetpack Navigation Component

-არქიტექტურული გადაწყვეტილებები
პროექტში გამოყენებულია MVVM პატერნი მონაცემებისა და UI ლოგიკის გამიჯვნისთვის:
1.Model: RecipeRepository მართავს მონაცემთა ნაკადს.
2.ViewModel: RecipeViewModel იყენებს combine ოპერატორს, რათა ერთდროულად დაამუშაოს ძებნის ტექსტი, კატეგორია და რეცეპტების სია.
3.View: ფრაგმენტები პასიურად „უსმენენ“ მონაცემების ცვლილებას, რაც უზრუნველყოფს UI-ს სტაბილურობას.

განსაკუთრებული ფუნქციონალი: Reactive Multi-Source Filtering
აპლიკაციაში რეალიზებულია მონაცემთა რეაქტიული დამუშავება. მომხმარებლის მიერ შეყვანილი ნებისმიერი სიმბოლო ან ფილტრის შეცვლა მომენტალურად აისახება ეკრანზე Flow.combine ფუნქციის მეშვეობით, რაც გამორიცხავს დაყოვნებას (Lag) და უზრუნველყოფს 0-მილიწამიან რეაგირებას.
